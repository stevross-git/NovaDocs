## Quick Start

```bash
# Clone and setup
git clone <repo-url>
cd novadocs
npm install

# Start development environment
npm run docker:dev
npm run dev
```

## Development

- Frontend: http://localhost:3000
- Backend API: http://localhost:8000
- GraphQL Playground: http://localhost:8000/graphql
- PostgreSQL: localhost:5432
- Redis: localhost:6379
